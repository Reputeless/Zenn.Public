## 1. 現行世代 Siv3D v0.6
2025 年 12 月現在、メインで利用されている Siv3D は v0.6 世代です。2021 年 9 月リリースの v0.6.0 に始まり、4 年にわたって多くの機能追加と改善が行われてきました。最新の v0.6.16 は過去リリースの中で最高クラスの安定性とユーザ数を誇るバージョンへと進化しました。

商業利用の面では、v0.6 世代で開発された『マクスウェルのパズルな悪魔』が 1 万本の販売を達成したほか、複数のタイトルが Steam で配信または配信予定となっていて、Siv3D はインディーゲーム開発における実用的な選択肢としての地位を築きつつあります。

教育の現場でも、Siv3D の採用は広がっています。上智大学、大阪電気通信大学、国際理工カレッジ、バンタンゲームアカデミーなど、さまざまな教育機関で講義や実習に使われており、Siv3D を通じて C++ やゲーム開発を学ぶ学生が増えています。とりわけ高専プロコン（競技部門）での採用率は極めて高く、過去 5 大会の優勝チーム中 3 チーム、準優勝チーム中 4 チームが Siv3D を採用するなど、学生エンジニアにとって重要なツールとなっています。


## 2. 次期メジャーアップデート Siv3D v0.8
v0.6 世代の 4 年間の間に C++ 言語自体も進化し、Siv3D の利用者層も拡大・多様化して、多くのフィードバックが寄せられる中、内部構造をゼロから刷新しないと対応が難しい課題や新機能の案が蓄積されてきました。そのため 2023 年からフルスクラッチでの Siv3D 新バージョンの開発を始めました。これが Siv3D v0.8 世代です。

### 2.1 開発リポジトリ
https://github.com/Siv3D/siv8

- Siv3D v0.8 の開発は、本家とは異なるリポジトリで進められています
- 現時点で Windows と macOS に対応していますが、ビルドは自前で行う必要があります
- `v8_develop_xxx` ブランチが最新の開発ブランチです


### 2.2 Siv3D v0.8 の更新内容

#### v0.6 との比較

| 項目 | v0.6.16 | v0.8.0（予定） |
|------|-----------|-----------|
| C++ のバージョン | C++17 および一部の C++20 | **C++20 および一部の C++23, C++26** |
| Visual Studio サポート | 2022, 2026 | **2026** |
| Xcode サポート | 14.3 以降 | **26.1 以降** |
| Linux サポート | Ubuntu 20.04 / 22.04 | **Ubuntu 24.04** |
| Windows 版アーキテクチャ | x86_64 | x86_64 |
| macOS 版アーキテクチャ | x86_64 | **ARM64 (Apple Silicon)** |
| Windows 版グラフィックス API | Direct3D 11 | Direct3D 11 |
| macOS 版グラフィックス API | OpenGL 4.1 | **Metal 3** |
| Linux 版グラフィックス API | OpenGL 4.1 | **OpenGL 4.5** |
| ヘッダのドキュメント | 主要な機能のみ | **ほぼ全機能を網羅** |

#### C++・全般
- Windows 版の `char` 文字列を UTF-8 として扱うように変更
- C++20 / 23 Ranges ライブラリを活用
- `Result<T, E>` 型の導入によるエラーハンドリングの改善
- 高速なメモリアロケータ mimalloc を導入（Windows 版）
- 主要クラスの JSON シリアライズ対応

#### レンダリングエンジン
- 乗算済みアルファを標準で採用
- BCn（DDS）テクスチャ形式をサポート（Windows 版）
- 複数のレンダーターゲット（MRT: Multiple Render Targets）対応
- Compute Shader を活用できる機能を追加

#### 2D 描画
- 新しい図形クラス `SuperEllipse` を追加
- グラデーション塗りつぶし、線の端点スタイルなどを拡充
- パターン（チェッカー、水玉模様、ストライプなど）で図形を塗りつぶす機能を追加
- 標準シェーダエフェクトを追加

#### 3D 描画
- 光源数と種類を拡充
- 標準機能でシャドウをサポート
- 新しいスカイボックス機能
- 水面エフェクト

#### 物理演算
- Box2D v3 への移行
- 3D 物理演算

#### フォント・絵文字描画
- 縦書き対応
- Variable Font をサポート
- COLRv1 フォントをサポート
- テキスト装飾を強化（より太いアウトライン、光彩、グラデーションなど）
- 任意のサイズの絵文字描画

#### GUI
- 10 種類以上の新しい SimpleGUI ウィジェットを追加
- SimpleGUI のカラー・形状カスタマイズ機能
- SimpleGUI の各種ウィジェットのクラス化

#### オーディオ
- オーディオの基本周波数を 44.1 kHz から 48 kHz に変更

#### 入力デバイス
- マルチタッチ対応（Windows 版）

#### ユーティリティー・その他
- ファイルアーカイブ / 暗号化をサポート
- OpenAI API 連係を拡充
- 標準フォントを MPLUS2 に変更
    
#### API の破壊的変更
- いくつかのクラスや関数名について、より技術的に正確で、伝わりやすい名前への変更を複数実施
- 例:
    - `BinaryRaader` → `BinaryFileReader`
    - `BinaryWriter` → `BinaryFileWriter`
    - `HashSet` / `HashTable` → `HashSet` / `HashMap`
    - `std::set` / `OrderedTable` → `BTreeSet` / `BTreeMap`
    - `FileSystem::ModulePath()` → `FileSystem::GetExecutablePath()`


## 3. これまでの実装状況・今後の予定

### 3.1 実装状況・今後の目標
- 各機能の実装状況と、2026 年 3 月までの目標です

| 機能 | 実装状況（2025 年 11 月） | 2026 年 3 月までの目標 |
|------|-----------:|----------------:|
| 基盤クラス（Array, String, Grid 等）| 80% | 90% |
| ウィンドウ関連 | 60% | 80% |
| 2D グラフィックス | 75% | 90% |
| 3D グラフィックス | 0% | 0% |
| 画像ファイル・画像処理 | 30% | 80% |
| フォント・テキスト描画 | 80% | 90% |
| GUI | 20% | 60% |
| オーディオ | 0% | 50% |
| 入力デバイス | 50% | 80% |
| ファイルシステム | 90% | 95% |
| ネットワーク・通信 | 10% | 50% |
| OpenAI API 連係 | 0% | 50% |
| 設定ファイル（JSON, INI, XML 等）| 40% | 80% |
| その他ユーティリティ | 10% | 30% |

- 3D グラフィックスは、前提とする諸機能の実装が完了した開発終盤に着手する予定です
- 今後もこの形式で、定期的に進捗を報告していく予定です


### 3.2 ロードマップの公開
- 2026 年 3 月をめどに、v0.8.0～v0.8.x の各リリースで追加する主要な機能のロードマップを公開します

### 3.3 Siv3D v0.8 体験版 SDK のリリース
- 開発中の Siv3D v0.8 を利用するには、2025 年 12 月現在、エンジンのソースコードを自前でビルドする必要があります
- 2026 年初頭に Windows 版と macOS 版の体験版 SDK をリリースする予定です
- 通常の Siv3D SDK と同様に、インストーラで簡単に導入できる形態で提供します

### 3.4 Siv3D v0.8 の完成時期
- 最も順調に開発が進んだ場合は、2026 年 7 月のリリースが目標です
- 現実的な予想としては 2026 年秋 ～ 2026 年末になる可能性が高いです

### Siv3D v0.6 世代の今後
- Siv3D v0.6 世代は、Siv3D v0.8 世代のリリースまで、引き続きバグ修正や小規模な機能追加を行いながらサポートを継続します。1 か月以内に v0.6.17 をリリースする予定です
- v0.8 世代で実装した機能を v0.6 世代にバックポートすることも検討しています

### 3.6 Siv3D v0.6 から v0.8 への移植
- 基本的には推奨しません。関数名やクラス名の変更だけでなく、文字描画のパラメータなどが大幅に変更されているため、v0.6 で実現していた表現を v0.8 で再現するのが難しい場合があります
- 前節で述べたように v0.6 世代のサポートは継続するため、可能であれば v0.6 世代で開発を続けることをお勧めします

### 3.7 Siv3D v1.0 に向けて
- Siv3D v0.8 は「Siv3D v1.0 の β 版」という位置づけです。v0.8 リリース後数か月かけて完成度を高めたうえで、Siv3D v0.8.x を Siv3D v1.0.0 とリネームする予定です
- v1.0 からは、破壊的な API 変更を極力避け、安定した API を提供することを目指します
